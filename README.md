# 雾雨之乡

> [雾雨之乡](https://vanillaaaa.org/)

## 项目简介

- 技术栈：Svelte、Vite、TypeScript

## 本地开发

1. 安装依赖：
   ```sh
   npm install
   ```
2. 启动开发服务器：
   ```sh
   npm run dev
   ```

## 构建与部署

- 生产环境构建：
  ```sh
  npm run build
  ```
- 预览生产环境：
  ```sh
  npm run preview
  ```

## 目录结构

- `src/`：源代码目录
- `build/`：构建输出目录
- `static/`：静态资源
